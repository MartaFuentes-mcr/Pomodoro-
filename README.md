# 🍅 Pomodoro Study Sanctuary

Aplicación de **temporizador Pomodoro** desarrollada con **React + Vite**, diseñada para crear un entorno de estudio relajante con música ambiental, fondos dinámicos y métricas de productividad.

🌐 **Demo en vivo**  
https://pomodoro-black-five.vercel.app

---

## ✨ Características

### ⏱️ Temporizador Pomodoro
- Temporizador configurable para:
  - sesión de concentración
  - descanso corto
  - descanso largo
- Seguimiento automático de ciclos

### 🎵 Música ambiental
- Reproductor integrado de música relajante
- Ideal para estudiar o trabajar con concentración

### 🌄 Fondos dinámicos
- Diferentes escenarios visuales
- Categorías de fondos
- Vista previa para elegir ambiente

### 📊 Métricas de productividad
- Conteo de pomodoros completados
- Seguimiento del progreso de estudio

### ✔️ Gestión de tareas
- Añadir tareas mientras estudias
- Organizar sesiones de trabajo

### ⚙️ Configuración personalizable
- Ajustar duración de sesiones
- Personalizar la experiencia de estudio

### 📱 Diseño responsive
- Adaptado para **ordenador y dispositivos móviles**

---

## 🛠️ Tecnologías utilizadas

- React
- Vite
- JavaScript
- CSS
- Custom React Hooks
