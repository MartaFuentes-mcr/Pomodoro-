# Study Sanctuary

Study Sanctuary es una aplicación **Pomodoro** desarrollada con **React + Vite** para mejorar la concentración durante el estudio o el trabajo.  
Incluye **fondos personalizables**, **duraciones configurables**, **ciclos**, y **música ambiental** para crear un entorno tranquilo y productivo.

## ✨ Características
- Modo Pomodoro, descanso corto y descanso largo
- Duraciones ajustables (timer, breaks y ciclos)
- Fondos personalizables (galería + URL + subir imagen)
- Música ambiental con control de volumen
- Atajos de teclado (iniciar/pausar, reset, música)
- Interfaz minimalista y responsive

## 🛠️ Tecnologías
- React
- Vite
- JavaScript
- CSS
